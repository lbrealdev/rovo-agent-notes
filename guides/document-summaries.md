# Document Summaries

Prompts for summarizing Confluence and AWS documentation.

**Use when:** You need to extract essential steps or understand documentation quickly.

---

## Summarize Documentation

Concise summary focusing on essential steps.

```text
Please provide a concise summary of the process outlined in this document, listing only the essential steps.
```

---

## Summarize Procedure

Summarize focusing only on step-by-step procedures.

```text
Summarize this document, focusing only on the step-by-step procedure for [specific task].
```
